## ページ構成

**1ページ完結LP**(スマホ最優先)。セクション案:

1. ファーストビュー(キャッチコピー + LINE CTA)
2. 悩み共感(仕事帰り・疲れ・休めていない)
3. コンセプト(上記コピー)
4. ひとやすみでできること(運動 × 対話)
5. トレーナー紹介
6. 料金(**全プラン明示**)
7. 体験の流れ
8. FAQ
9. アクセス・店舗情報
10. クロージング LINE CTA(＋追従CTAボタン)

## デザイントーン(2026-06-12 改訂: 立て看板PDF準拠)

- **シック・高級感**: チャコールブラウン × 生成り × 深紅の差し色(立て看板のトーン&マナーに統一)
- フォントは**明朝体**(Shippori Mincho)。字間広め、角丸最小限
- ヒーローは**写真主役**: 暗めの店内写真を全面に、文字は小さくふわっと順送りフェードイン
- 当初の「ナチュラル・温かみ系(緑×テラコッタ)+丸ゴシック」は看板とのトーン不一致のため廃止
- 写真は**実写真**を使用(店内・設備・トレーナー)

## 技術・運用

- 実装: 素の HTML + CSS + 少しの JS(ビルド不要・依存ゼロ)
- ホスティング: **GitHub Pages**(要 git 初期化 + GitHub リポジトリ作成)
- ドメイン: まず `xxx.github.io` で公開 → 軌道に乗ったら独自ドメイン移行
  (移行時に SEO 評価がリセットされる点は了承済み)
- 更新運用: ほぼ固定・たまに修正。CMS なし。日々の発信は LINE / SNS 側で行う

## 実装済みの追加判断(3者レビュー反映)

- CTA は5接点(ヒーロー/できること後/料金後/クロージング/追従)。文言は「LINEで体験を予約する」+「無理な勧誘はしません」
- テラコッタはテキスト禁止(コントラスト AA 不達)。テキスト強調は暗色 #A85D38
- LINE 緑上の白文字は AA 不達だが LINE ブランドガイド準拠を優先する意図的例外
- JSON-LD は仮データ公開を避けるためコメントアウト(実データ反映時に有効化)
- 地図は iframe 直埋めせず「Googleマップで開く」リンク(ファサードパターン)
- 計測: GA4 で CTA 位置別クリック+スクロール深度(js/script.js)
