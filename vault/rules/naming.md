# ファイル命名規則

このVault内のすべてのファイル・フォルダは以下のルールに従って命名する。

---

## ファイル命名パターン

| ノート種別 | 命名パターン | 例 |
|---|---|---|
| デイリーノート | `YYYY-MM-DD.md` | `2026-06-14.md` |
| 記事下書き | `記事_タイトル略称.md` | `記事_コンプレックス克服.md` |
| アイデア | `アイデア_テーマ.md` | `アイデア_朝活習慣.md` |
| リサーチ | `リサーチ_テーマ.md` | `リサーチ_自己肯定感研究.md` |
| プロジェクト | `PJ_プロジェクト名.md` | `PJ_フィットネスアプリ.md` |
| ミーティング | `MTG_相手_YYYY-MM-DD.md` | `MTG_山田太郎_2026-06-14.md` |

---

## フォルダ命名規則

フォルダ名はすべて**英小文字**を使用する：

| フォルダ | 用途 |
|---|---|
| `daily/` | デイリーノート |
| `articles/` | 記事下書き |
| `ideas/` | アイデアメモ |
| `research/` | リサーチノート |
| `projects/` | プロジェクト管理 |
| `rules/` | Vault運用ルール |
| `templates/` | テンプレート |
| `wiki/` | 索引・用語集 |
| `MOC/` | Map of Content（例外：大文字） |

---

## その他のルール

- ファイル名にスペースは使用しない（アンダースコア `_` を使う）
- 日本語ファイル名はOK（テンプレートや記事下書きなど）
- 拡張子は必ず `.md`
- 重複を避けるため、作成前に既存ファイルを確認すること
